# 🌌 Alfonso Zendejas — 3D Cinematic Developer Portfolio

Welcome to the source code of my personal 3D developer portfolio — a space-inspired, immersive experience built with **React Three Fiber**, featuring a dynamic **black hole** scene, camera transitions, and an interactive UI designed to showcase not just code, but **vision**.

---

## 🚀 About This Project

This portfolio is not a static resume — it's an **interactive journey**.

- 🌑 A rotating **black hole** acts as the central visual metaphor for my focus, depth, and gravity as a developer.
- 🪐 A cinematic **camera transition** reveals a planet orbiting the black hole, where I share my background, virtues, and mission.
- ✨ The experience blends **3D realism**, dynamic lighting, and UI overlays to present myself as a **full-stack developer** with 3D capability, vision, and philosophical grounding.

---

## 🧩 Tech Stack

| Layer        | Tools                                         |
|--------------|-----------------------------------------------|
| 3D Engine    | [React Three Fiber](https://docs.pmnd.rs)     |
| Assets       | Blender (.glb export)                         |
| Animations   | GSAP / useFrame / useSpring                   |
| Lighting     | Custom cinematic setup with `three.js` lights |
| Styling      | Tailwind CSS                                  |
| UI/UX        | React components with floating overlay text   |

---

## 🎯 Features

- ✅ Real-time rendering of a high-fidelity black hole model
- ✅ Cinematic lighting (warm/cool contrast)
- ✅ Dynamic camera movement (scroll or button-based)
- ✅ Custom overlay text for personal introduction and skills
- ✅ Space backdrop with particles or HDRI environment

## 📦 Installation

```bash
git clone https://github.com/alfonsozendejas/3d-portfolio.git
cd 3d-portfolio
npm install
npm run dev
```
